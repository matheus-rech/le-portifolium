## Decisions

We agreed on the dark theme approach for the gallery. Key points:

- Gold accent (#c9a96e) as the primary highlight color
- IBM Plex Sans for body text, DM Serif Display for headings
- Card-based layout with iframe previews for HTML artifacts
- Subfolder support for organizing by project or category
