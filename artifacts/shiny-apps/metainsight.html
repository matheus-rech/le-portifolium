MetaInsight

An interactive web tool for Network Meta-Analysis (NMA) that leverages established analysis routines. Built by CRSU-Apps.

Shiny R Network Meta-Analysis shinydashboard shinyjs

Features

  • Load and validate NMA datasets
  • Run pairwise and network meta-analysis
  • Interactive forest plots and network graphs
  • Ranking panels with SUCRA scores
  • Comprehensive user guide and troubleshooting

Source

github.com/CRSU-Apps/MetaInsight

R -e "shiny::runApp('~/Documents/GitHub/MetaInsight')"
