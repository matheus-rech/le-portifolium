AI Library -- R Code Executor

A Shiny server for remote R code execution. Upload R scripts, execute them, view output and plots, and download reports.

Shiny R rmarkdown

Features

  • Upload and execute R scripts remotely
  • Live console output and plot rendering
  • Download generated reports
  • Simple file upload interface

Run Locally

R -e "shiny::runApp('~/Documents/AI Library/Code')"
