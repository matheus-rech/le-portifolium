Meta-Analysis Quick Calculator

DerSimonian-Laird random-effects meta-analysis with forest plots. Uses Gradio UI (Python) powered by R's meta package via reticulate.

Gradio R + meta reticulate

Features

  • Upload CSV with study data (study, mean1, sd1, n1, mean2, sd2, n2)
  • Automatic DerSimonian-Laird random-effects model
  • Forest plot generation
  • Gradio web UI via Python-R bridge

Run Locally

cd ~/Documents/gradio-meta-r source .venv/bin/activate Rscript app.R
