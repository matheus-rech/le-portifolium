Cochrane Meta-Analysis Report

Parameterized Quarto report template for Cochrane-style meta-analysis. Generates executive summaries with forest plots, funnel plots, heterogeneity stats, and interpretation. Outputs to HTML (cosmo theme) and PDF.

Quarto (.qmd) R + knitr Cochrane embed-resources

Parameters

ParamDescription
n_studiesNumber of included studies
interventionIntervention name
comparisonComparator name
pooled_orPooled Odds Ratio
I2Heterogeneity (I-squared)
forest_plot_pathPath to forest plot image
funnel_plot_pathPath to funnel plot image

Report Sections

  • Executive Summary with key finding callout
  • Methods and statistical approach
  • Forest plot and funnel plot visualization
  • Heterogeneity assessment (I2, Q, tau2)
  • Interpretation and clinical significance

Render

quarto render generate_report.qmd \ -P intervention:"Drug A" \ -P comparison:"Placebo" \ -P n_studies:12 \ --to html
